# Git Branch 

## 1. To see list all branches on your repository.
```bash 
git branch
```



## 2. Create a new branch.
```bash 
git branch <branch_name>
```
#### Example:
```bash 
git branch dev_rakib
```



## 3. Delete a merged branch.
```bash 
git branch -d <branch_name>
```
#### Example:
```bash 
git branch dev_rakib
```
